# 公众科普网页内容包（可直接上线的静态站点）

本包将论文《Towards a neuroimaging biomarker for predicting cognitive behavioural therapy outcomes in treatment-naive depression: Preliminary findings》（Psychiatry Research, 2023）转写为公众可读的网页，并提供可落地的交互模块与数据结构。  
所有关键断言均在页面中以“证据锚点（页码/图号/节）”标注来源（证据：论文PDF第1–3页，图1）。

---

## 目录结构

- `index.html`：主页面（中文）
- `css/style.css`：样式
- `js/app.js`：前端渲染与交互逻辑（无第三方依赖）
- `data/content.json`：页面文案与证据锚点（开发可直接消费）
- `data/interactive.json`：交互模块配置（时间轴、图1热区、转录组探索器）
- `assets/figure1.png`：论文图1原图（未改动内容，来自论文第3页图1）
- `assets/diagram_*.svg`：本包新增概念信息图（不包含论文未报告的数值）

---

## 本地预览（推荐）

部分浏览器会阻止 `file://` 方式读取 JSON。请在本目录运行本地服务器：

```bash
python -m http.server 8000
```

然后在浏览器打开：

```text
http://localhost:8000
```

---

## 页面包含的三类互动

1) **8周治疗旅程时间轴**：按“基线→CBT→治疗后→分析”理解研究设计（证据：第2页2.1–2.4节；第2页4.1节）。  
2) **图1互动导览**：不修改原图文件，通过热区解释 A–J 子图的核心信息与关键数字（证据：第3页图1图注）。  
3) **影像转录组探索器**：通路分组切换 + 细胞类型条形图（仅使用图注给出的重叠基因数与p值）（证据：第3页图1I–J图注；第2页2.4节）。

## 许可与合规提示

- 论文为开放获取 **CC BY-NC-ND 4.0**（证据：论文第1页页眉许可说明）。  
- 本包对论文图像的使用：仅复用论文图1原图文件 `assets/figure1.png`，并在页面中注明来源与许可；其余图为本包原创概念图（不重绘论文原图、不引入论文未报告数值）。

---

## 数据边界（重要）

- 论文未提供个体级 HAMD 或 ReHo 原始数据，因此本包的交互与信息图不会生成“个体曲线复现”或“在线预测器”。  
- 机器学习部分仅展示论文报告的误差指标（MAE=4.44、MSE=33.88）与方法（SVR、留一交叉验证）（证据：第2页2.3节与4.2节；图1D）。
