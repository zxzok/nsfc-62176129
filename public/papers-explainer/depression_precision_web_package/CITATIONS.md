# 证据锚点速查（来自论文原文）

> 说明：下列“页码/图号/表号”对应本内容包所引用的关键证据位置，便于审核与追溯。

## 1）研究动机与现实背景
- 抑郁终生患病率 15%–18%，约3.5亿人受影响；单药3–6个月缓解率最多约40%；全球负担30年未明显下降（证据：第1页 Background）

## 2）研究设计（两中心两阶段）
- 两阶段两中心总体流程（证据：图1，第3页）
- 中心1：238 MDD、66 HC；中心2：72住院青少年/青年（38 MDD、34 双相II型抑郁发作）；入组阈值（证据：第2页 2.1；第1页 摘要Methods）
- 分型方法：t‑SNE降维 + GMM聚类 + 一致性聚类稳定性检验 + SVM评估（证据：第2–3页 2.3）
- 选靶点方法：AAL‑90脑区、逐脑区分类器、Top‑10候选脑区分布（证据：第3页 2.4；第4页 3.2；图3，第5页）
- rTMS方案：原型10Hz/1200脉冲；非典型1Hz/1000脉冲；共20次/两周；100%RMT（证据：第3页 2.5）

## 3）关键结果
- 两亚型：原型(n=143)与非典型(n=95)；前后失衡方向相反；稳定性m1=0.93、m2=0.73；分型后模型性能更高（F1/AUC）且置换检验p<0.002（证据：第4页 3.1；图2）
- 靶点：原型→dmPFC；非典型→OCC；候选脑区主导分布“6/10”（证据：第4页 3.2；图3）
- 临床应用：中心2分型原型16、非典型56；完成两周者10与41；无严重不良事件；多数影像变化发生在第1周（证据：第4–5页 3.3）
- T2总体响应率：原型90.00%、非典型70.73%；T2缓解率：原型30.00%、非典型46.34%；自杀意念条目改善百分比（证据：第5页 3.4；图4B，第6页）

## 4）局限性与下一步
- 无假刺激对照；参与者未完全盲法；样本量较小；可能需要更长疗程；呼吁更大样本双盲随机对照试验，并提到NCT05465928（证据：第7页 Discussion；第7页 Conclusions）
